Personal portfolio and blog of **Anirudh Negi** — Software Engineer III building reliable backend platforms, AI-ready data systems, and developer tooling at scale.

## About

This site showcases my work, experience, and exploration at the intersection of backend engineering and modern AI capabilities. It is built with plain HTML/CSS/JS and hosted on GitHub Pages using the [Cayman](https://github.com/pages-themes/cayman) Jekyll theme.

## Contents

- **About** — Who I am and what I care about
- **Impact** — Stories from building systems at scale
- **Experience** — Work history and roles
- **Projects** — Live GitHub projects, filterable by language
- **AI Projects** — Projects built with AI pair programming (Claude Code, Qwen, LangChain)
- **Case Studies** — Deep dives into systems, data, and AI workflows
- **Skills** — Technologies I enjoy working with
- **Education** — Academic and continuous learning background
- **Contact** — Get in touch

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- [Jekyll](https://jekyllrb.com/) + GitHub Pages
- Cayman theme (`_config.yml`)
- Google Fonts (JetBrains Mono, Space Grotesk)

## Local Development

1. Install [Ruby](https://www.ruby-lang.org/) and [Bundler](https://bundler.io/)
2. Run `bundle exec jekyll serve`
3. Open `http://localhost:4000` in your browser

Alternatively, open `index.html` directly in a browser for a quick preview.
